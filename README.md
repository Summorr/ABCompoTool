**A tool for composprites of AngryBirds**

## ✨ Features
- ✅ Import Composprites
- ✅ Export Composprites to png
- ✅ Multi-language[en and zh]

## 📦 How to use
### First you need [KA3DConvert by GZHYBFHHJ](https://github.com/GZHYBFHHJ/KA3DConvert) to Convert .dat to .xml to import to the program
